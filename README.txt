# BlueHome Backend

Este backend incluye:
- Conexión con OpenAI API (GPT-4 o 3.5 turbo).
- Cálculo automático de tarifas de administración (10.5% + IVA + 2.05% de amparo básico).
- Notificaciones internas al área comercial o de arrendamientos.
- Consulta de inmuebles desde Google Sheets.
- Respuesta automática al mostrar interés en dejar inmueble o arrendar.
- Información de contacto general y de estudios de arrendamiento.
